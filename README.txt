ABOUT:
 Have you ever been in an area with a lot of endermen, and the only option
 is to wear a pumpkin, obscuring your field of view? Now you can avoid that!
 This pack is designed for minecraft bedrock 1.16. It should work on older 
 versions, as well as newer ones. 

HOW TO IMPORT
 1. Download .mcpack file
 2. Import .mcpack file into your client's rescource pack directory
 3. Open Minecraft and select NoPumpkinBlur
 4. Now you can use it!
